# Технічні специфікації — Котедж Куземський

> Всі рішення нижче — **погоджені та затверджені**.
> Останнє оновлення: 11.06.2026

---

## ⚡ ЕЛЕКТРИКА

### Кабелі (затверджено 01.05.2026)
| Лінія | Перетин | Тип |
|-------|---------|-----|
| Освітлення | 3×1.5 мм² | Мідь, 3 жили (L, N, PE) |
| Розетки | 3×2.5 мм² | Мідь, 3 жили (L, N, PE) |
| Потужні прилади (духовка, котел тощо) | окрема лінія, більший перетин | Мідь, 3 жили |

- Марка кабелю: ВВГнг-LS або аналог (⚠️ підтвердити в електрика)
- Всі з'єднання ТІЛЬКИ в розподільчих коробках через клеми (WAGO), скруток у стінах — немає

### Щиток
- Окремі автомати на кожну групу: освітлення, розетки, кухня, котел, техніка тощо
- Захист через УЗО або дифавтомати на кожній групі
- У всіх розетках — підключене заземлення (PE)

### Smart Home — підготовка
- Освітлення керується через **реле в електрощиті** (не через вимикачі напряму)
- Реле: **Zigbee-сумісні** або аналогічний протокол
- **Зіркова схема** від щитка до кожного світильника
- **6 груп освітлення** (1-й поверх) — окремі керуючі лінії
- До місця майбутньої LED панелі прокласти:
  - фаза (L)
  - нуль (N)
  - 6 сигнальних жил (по одній на групу)
  - резерв 1-2 жили
- Наразі реалізація через реле в щитку; панель — у майбутньому без переробки проводки
- **Майстер-вимикач** для всього будинку

### Сходи — підсвітка
- Підсвітка **кожної сходинки** (окремий вивід під кожну)
- Одне **настінне бра** на основній стіні сходів
- Стельового освітлення на сходах НЕ ПЕРЕДБАЧЕНО — тільки підсвітка + бра

### Висоти встановлення (стандарт)
- Вимикачі без позначення висоти: **h = 900 мм** від підлоги
- Розетки без позначення висоти: **h = 300 мм** від підлоги
- Розетки/виводи біля дверей: на відстані **200 мм** від прорізу

### Зовнішні розетки
- Розетки на вулиці (для гірлянд, зовнішніх приладів) — керовані з **вимикача в прихожій**

## 🚿 САНТЕХНІКА

### Унітази (обидві ванні)
- Тип: **підвісні, з інсталяцією**
- Висота чаші: **42–43 см** від чистої підлоги
- Кнопки змиву: **з плитки** (без окремих люків для доступу)
- Доступ для обслуговування: тільки через кнопку змиву

### Гігієнічний душ (обидва санвузли)
- Монтаж: **прихований** (змішувач у стіні)
- Окремий змішувач (гаряча + холодна)
- Вихід шланга зі стіни
- Клапан перекриття
- Висота: **~70 см** від підлоги, біля унітазу

### Душова система (душова зона, 1-й поверх)
- **Тропічний душ** (rain shower)
- **Ручний душ**
- **Перемикач** між ними
- **Термостатичний змішувач**
- Монтаж: **прихований**
- Висоти монтажу:
  - Змішувач: **110–120 см** від підлоги
  - Тропічний душ: **210–220 см**

### Душовий трап
- Тип: **лінійний**, розміщення біля стіни
- Сифон + зворотний клапан
- Можливість ревізії/чистки **зверху**

### Каналізаційні труби
- **Шумоізоляція** труб (або шумоізоляційні труби)
- Особливо важливо між поверхами (не чути злив)

### Вологозахист
- **Гідроізоляція душових зон**: підлога + стіни (обов'язково)
- **Тепла підлога** у всіх мокрих зонах

### Рушникосушки
- Тип: **комбінована** (водяна + електрична) або електрична

### Пральна машина + сушарка
- Розміщення: **одна над одною** (2-й поверх, ванна)

### Вода
- Розводка **прихована**, але з **ревізіями** в потрібних місцях
- **Фільтрація води** (система очищення — кухня)
- **Підігрів гарячої води: котел** (бойлер окремий — не потрібен, котел гріє воду)

## 🔥 ОПАЛЕННЯ

### Тепла підлога
- Тип: **водяна електрична** (контур теплої підлоги + електрокотел)
- Охоплення: **1-й і 2-й поверхи**
- Мокрі зони — тепла підлога обов'язково
- Потужність: **15 кВт, три фази по 5 кВт** (погоджено з Миколою 11.06.2026)

### Котел
- Тип: **електричний** (вже встановлений)
- Функція: нагрів підлоги + підготовка гарячої води (ГВП)

### Терморегулятори
- ⚠️ **Рішення не прийнято**: окремі по кімнатах чи централізовано?

## 🏠 СТЕЛІ

### 1-й поверх
- Натяжна стеля з **LED стрічкою в стельовому профілі** (з розсіювачем)
- Відступ від конструкції: **6 см**

### 2-й поверх
- Утеплення мінеральною ватою + натяжна стеля
- Мансардні вікна: каркас + зашиття гіпсокартоном навколо

### Горище (над кімнатою 1.8 + частина коридору)
- Призначення: комора / технічний простір
- Ширина: ~5.63 м; глибина: кімната 1.8 (~2.3 м) + частина коридору (⚠️ уточнити)
- Доступ: люк + розкладні сходи; люк знаходиться у коридорній частині горища
- Покрівля: моноухил — h=0 на стіні де закінчується спальня master (1.7), рівномірно зростає через коридор, пік над кімнатою 1.8 (зовнішня стіна)
- Непридатна зона: частина горища над коридором де h < 1 м (ближче до стіни спальні)
- Придатна зона для зберігання: над кімнатою 1.8 (пік стелі)
- Освітлення: 2 LED світильники у зоні піку (над кімнатою 1.8) — лівий і правий
- Вимикач: права стіна коридору 2-го поверху, біля сходів, разом з іншими вимикачами
- Розетки: 2 шт. на лівій бічній стіні (зона піку) + 2 шт. на правій бічній стіні (зона піку)
- Кабелі: освітлення 3×1.5 мм², розетки 3×2.5 мм²
- Щиток: 1 автомат освітлення + 1 автомат розетки
- ⚠️ Виміряти висоту в піку (зовнішня стіна кімнати 1.8) — для вибору типу LED кріплення

## 🏗️ КОНСТРУКТИВ (що вже є в коробці)

| Елемент | Специфікація |
|---------|-------------|
| Зовнішні стіни | Блок відсівки |
| Фасад | Утеплення пінопластом + декоративне оздоблення |
| Перекриття | Залізобетонні плити |
| Дах | Металочерепиця |
| Внутрішні стіни | Штукатурка + армування сіткою + шпаклівка + фарба |
| Вікна | Металопластик, 5-камерний профіль, енергоефективний склопакет |
| Двері | Монтажний комплект (вхідні + міжкімнатні) |
| Сходи | Металевий каркас + чистова обробка (тверді породи) + перила |
| Підлога | Тепла підлога (водяна) + ламінат/керамічна плитка |
| Стелі 1-й пов. | Натяжна стеля |
| Стелі 2-й пов. | Мінвата + натяжна стеля |
| Каналізація | Локальна (кільця) + пластикові труби до будинку + розводка |
| Електрика | Підведена до лічильника + розводка по будинку |
| Опалення | Електрокотел + водяна тепла підлога |
| Водопостачання | Лічильник + технічні труби |

### Утеплення даху (поточний стан)
- Крокви: ~180 мм
- Утеплювач: 150 мм (покладено)
- Вентиляційний зазор під мембраною: ~30 мм
- ⚠️ **Питання:** додавати перехресний шар 50 мм під кроквами? (опускає стелю на 5 см)

## 📐 ПІДЛОГОВІ МАТЕРІАЛИ (зонування)

| Зона | Матеріал |
|------|----------|
| Кухня | Плитка / керамограніт |
| Вітальня | Паркет |
| Прихожа | Плитка / керамограніт (вологостійке, неслизьке) |
| Санвузли | Керамічна плитка |
| Сходи | Тверді породи дерева |

- Перепад рівня підлоги: **не більше 2 мм/м**
- На переходах між матеріалами: **розширювальні шви або декоративні вставки**
- Висоту стяжки визначити **по факту** вибору покриттів
