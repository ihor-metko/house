Електрика та освітлення – перший поверх

Вулиця

Плануються гірлянди, тому потрібно передбачити вимикач у прихожій, який керує групою розеток на вулиці.

Потрібно кілька розеток для підключення всіх зовнішніх приладів.

Прихожа

Вимикач для розеток на вулиці (для гірлянд).

Поличка для дрібниць повинна мати розетку і LED-підсвітку.

Сенсорне LED-управління для підсвітки.

Додати розетку біля пуфу.

Якщо дзеркало буде на дверях, додаткове освітлення над полицею для дрібниць не потрібне; якщо дзеркала не буде — підсвітку слід передбачити.

Ванна (перший поверх)

Все за попереднім планом — готово.

Кімната під сходами

Світло має бути на протилежній стороні, щоб дальня зона була освітлена.

Додати більше розеток у різних місцях для зручності.

Стіна між сходами і прихожою

Секція котячого лотка:

Дві низькі розетки для лотка та додаткової техніки.

Дві високі розетки для Wi-Fi роутера та резервного живлення (UPS), щоб працювали незалежно.

Секція робота-пилососа: дві низькі розетки біля підлоги для док-станції.

Секція котла та обладнання: три високі розетки для котла, насосів та іншої техніки; обов’язково окрема лінія з захистом.

Кухня

Основне стельове освітлення + люстра над столом — один димер для регулювання яскравості.

Біля фартуха додати USB-A або Type-C порти для зарядки телефонів, планшетів та навушників.

Розетки на фартусі винести зверху для зручного підключення техніки.

Вітальня

Біля дивану: USB-A або Type-C для зарядки гаджетів з обох боків дивана, що дозволяє обійтися однією комбінованою розеткою.

Біля телевізора:

USB та Type-C для медіапристроїв, стрімінгових коробок або телевізора.

Кілька LAN-портів (RJ45) для телевізора, приставок та ігрових консолей.

Вимикачі біля сходів

Використати сенсорні або мультигрупові вимикачі, щоб не захаращувати стіну великою кількістю клавіш.

Керування має включати:

Люстру вітальні, точкове освітлення вітальні та стельове світло.

Люстру над столом, точкове освітлення кухні та освітлення фартуха.

Світло на сходах і бра на сходах.

Прохідний принцип: вимикач біля сходів вмикає і вимикає всі ці групи, а на другому поверсі світло можна вимкнути з того ж блоку.

Сходи

Підсвітка кожної сходинки.

Настінне бра на основній стіні.

Стельове освітлення на сходах не передбачене — світло лише від підсвітки і бра.

Розетки на сходах не потрібні, якщо немає спеціальної потреби для техніки.

План освітлення та розеток – другий поверх котеджу
Коридор

Додати вимикач освітлення біля спальні.

Ванна кімната (другий поверх)

Основне освітлення: стельове, по всій площі.

Вимикач у ванній не потрібен, оскільки:

Один вимикач ззовні керує точковим освітленням;

Інший вимикач ззовні керує додатковим освітленням (над ванною, під ванною, дзеркало над умивальником).

Розетки:

Зона пральної машини + сушарки + полички: 3 розетки 220V.

Зона умивальника: 2 розетки 220V + 1 USB-A або Type-C.

Ванна кімната (перший та другий поверхи)

Обладнати гігієнічним душем.

Спальня

Основне освітлення: стельове, по всій площі.

Біля телевізора:

Вивід під інтернет;

Розетки Type-C та USB.

Гардеробна (другий поверх)

Передбачити мінімум 2 розетки 220V.

Кабінет

Освітлення:

Стельове;

Настінне на перегородці робочої зони (вмикається через подвійний вимикач).

Настінна бра на основній стіні, що вмикається разом із робочою зоною.

Розетки:

Робоча зона: 4 розетки + USB / Type-C + вивід під інтернет.

По одній розетці в кожному кутку кімнати.

Кімната справа

Освітлення:

Стельова підсвітка (двійний вимикач);

Точкове світло по периметру, розділене на 2 незалежні групи.

Розетки:

Біля вікна – для гірлянд.

За телевізором – розетки внизу (щоб кабелі не звисали та не заважали).

Загальні моменти по всьому дому

Майстер-вимикач для освітлення всього будинку.
